TOP - Recipes Project

This is the first real project in The Odin Project's extense curriculum. It is about building a website on recipes using HTML.
The aim of the project is consolidating and demonstrating what we've learnt about HTML (images, links, types of links, tags, attributes, etc.)

Basically, we build a website of recipes that contains links that refer to specific recipes (in my case waffles, risotto, and tacos).
